# IOI Preparation To-Do List

This to-do list breaks down the syllabus into actionable steps. Mark items as complete as you master them. Refer to `resources.md` for in-depth materials.

---

## Part 1: Beginner Foundations

### Module 1: C++ Fundamentals
- [ ] Read through Chapters 0-2 on [learncpp.com](https://www.learncpp.com/).
- [ ] Read the "Basics of C++" section on [cplusplus.com](https://www.cplusplus.com/doc/tutorial/).
- [ ] **Practice:** Write simple programs that take input and produce output.

### Module 2: Control Structures & Functions
- [ ] Read through Chapters 4-5 on [learncpp.com](https://www.learncpp.com/).
- [ ] **Practice:** Solve problems involving loops, conditionals, and functions on a platform like Codeforces.

### Module 3: Basic Data Structures
- [ ] Learn about Arrays, Strings, Vectors, Pairs, and Tuples from [GeeksforGeeks](https://www.geeksforgeeks.org/data-structures/).
- [ ] **Practice:** Solve problems that require storing and manipulating collections of data.

### Module 4: Standard Template Library (STL) - Core
- [ ] Study `std::vector`, `std::string`, `std::algorithm`, `std::set`, and `std::map` from the [cplusplus.com STL Reference](http.www.cplusplus.com/reference/stl/).
- [ ] **Practice:** Use these STL containers to solve problems on CSES.

### Module 5: Complexity Analysis
- [ ] Understand Big O notation from [GeeksforGeeks](https://www.geeksforgeeks.org/analysis-of-algorithms/).
- [ ] **Practice:** Analyze the time and space complexity of the code you write.

### Module 6: Introductory Algorithms
- [ ] Learn about Searching and Sorting algorithms.
- [ ] **Practice:** Solve introductory problems on [CSES](https://cses.fi/problemset/) and participate in [Codeforces](https://codeforces.com/) Div. 3 & 4 contests.

---

## Part 2: Intermediate Skills

### Module 7: Advanced Sorting & Searching
- [ ] Study Merge Sort, Quick Sort, and other advanced sorting techniques on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve problems that require efficient sorting or searching.

### Module 8: Core Data Structures
- [ ] Learn about Stack, Queue, Deque, Priority Queue, and DSU on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve problems on platforms like AtCoder and USACO Bronze/Silver.

### Module 9: Introduction to Graph Theory
- [ ] Study graph representations, BFS, and DFS from [TopCoder](httpss://www.topcoder.com/thrive/tracks/graph-theory) or [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve graph traversal problems.

### Module 10: Introduction to Dynamic Programming (DP)
- [ ] Learn the fundamentals of DP from [TopCoder](httpss://www.topcoder.com/thrive/tracks/dynamic-programming).
- [ ] **Practice:** Solve classic DP problems on CSES.

### Module 11: Greedy Algorithms
- [ ] Understand the greedy approach from [GeeksforGeeks](https://www.geeksforgeeks.org/greedy-algorithms/).
- [ ] **Practice:** Solve greedy problems and understand why the greedy choice works.

### Module 12: Basic Number Theory & Math
- [ ] Study prime numbers, GCD/LCM, and modular arithmetic on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve math-related problems on Codeforces.

---

## Part 3: Advanced & Expert Topics

### Module 13: Advanced Data Structures
- [ ] Study Segment Trees, Fenwick Trees, and Tries on [CP-Algorithms](https://cp-algorithms.com/) and [USACO Guide](https://usaco.guide/).
- [ ] **Practice:** Solve problems requiring these data structures on Codeforces Div. 1 and TopCoder.

### Module 14: Advanced Graph Algorithms
- [ ] Learn about Shortest Path algorithms, MST, SCC, and LCA on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Tackle advanced graph problems on AtCoder and USACO Gold/Platinum.

### Module 15: Advanced Dynamic Programming
- [ ] Study DP on Trees, Bitmask DP, and other advanced DP techniques from [Codeforces Blogs](https://codeforces.com/blog/entry/43256).
- [ ] **Practice:** Solve challenging DP problems from the CSES DP section.

### Module 16: String Algorithms
- [ ] Learn KMP, Z-Algorithm, and Hashing on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve string processing problems on various platforms.

### Module 17: Computational Geometry
- [ ] Study basic computational geometry concepts on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve geometry problems, paying close attention to edge cases and precision issues.

### Module 18: Advanced Mathematical Concepts
- [ ] Learn Combinatorics, Game Theory, and FFT on [CP-Algorithms](https://cp-algorithms.com/).
- [ ] **Practice:** Solve problems on Project Euler and from advanced contest rounds.

### Module 19: Theoretical Computer Science
- [ ] Understand Amortized Analysis, especially for data structures. Refer to the CLRS book or relevant [Codeforces Blogs](https://codeforces.com/search?query=amortized%20analysis).
- [ ] Learn the basics of Finite State Machines and their applications in string problems.
- [ ] **Practice:** Find problems that can be modeled with automata or require understanding of amortized complexity.

### Module 20: Miscellaneous Techniques
- [ ] Read about techniques like Square Root Decomposition from [Codeforces Blogs](https://codeforces.com/blog/entry/79102).
- [ ] **Practice:** Find problems that can be solved with these specific techniques.
