# IOI Preparation Syllabus: From Beginner to Expert

This syllabus provides a structured pathway for preparing for the International Olympiad in Informatics (IOI), designed to take a student from a beginner to an expert level.

---

## Part 1: Beginner Foundations

This section focuses on building a strong foundation in C++ and fundamental problem-solving techniques.

### **Module 1: C++ Fundamentals**
*   **Topics:** Basic syntax, data types, variables, operators, input/output (I/O) streams, basic file I/O.
*   **Resources:** [learncpp.com](https://www.learncpp.com/) (Chapters 0-2), [cplusplus.com Tutorial](https://www.cplusplus.com/doc/tutorial/)

### **Module 2: Control Structures & Functions**
*   **Topics:** Conditionals (if/else), loops (for/while), functions, scope, recursion.
*   **Resources:** [learncpp.com](https://www.learncpp.com/) (Chapters 4-5)

### **Module 3: Basic Data Structures**
*   **Topics:** Arrays (1D/2D), strings (`std::string`), `std::vector`, `std::pair`, `std::tuple`.
*   **Resources:** [GeeksforGeeks: Data Structures](https://www.geeksforgeeks.org/data-structures/)

### **Module 4: Standard Template Library (STL) - Core**
*   **Topics:** `std::vector`, `std::string`, `std::algorithm` (sort, binary_search, min/max), `std::set`, `std::map`.
*   **Resources:** [cplusplus.com STL Reference](http.www.cplusplus.com/reference/stl/)

### **Module 5: Complexity Analysis**
*   **Topics:** Big O notation, analyzing time and space complexity of simple algorithms.
*   **Resources:** [GeeksforGeeks: Analysis of Algorithms](https://www.geeksforgeeks.org/analysis-of-algorithms/)

### **Module 6: Introductory Algorithms**
*   **Topics:** Linear Search, Binary Search, Bubble Sort, Insertion Sort, Selection Sort, basic recursion.
*   **Practice:** Solve problems in the "Easy" or "A" category on Codeforces, CSES introductory problems.

---

## Part 2: Intermediate Skills

This section builds on the foundation with more complex data structures and algorithms.

### **Module 7: Advanced Sorting & Searching**
*   **Topics:** Merge Sort, Quick Sort, Heap Sort, Ternary Search, Custom comparators for `std::sort`.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/)

### **Module 8: Core Data Structures**
*   **Topics:** Stack, Queue, Deque, Priority Queue (Heap), Disjoint Set Union (DSU).
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/), TopCoder Tutorials

### **Module 9: Introduction to Graph Theory**
*   **Topics:** Graph representation (Adjacency Matrix/List), Breadth-First Search (BFS), Depth-First Search (DFS), Connected Components, Topological Sort.
*   **Resources:** [TopCoder: Graph Theory](httpss://www.topcoder.com/thrive/tracks/graph-theory)

### **Module 10: Introduction to Dynamic Programming (DP)**
*   **Topics:** Memoization vs. Tabulation, classic 1D and 2D DP problems (Fibonacci, Knapsack, Longest Common Subsequence).
*   **Resources:** [TopCoder: Dynamic Programming](httpss://www.topcoder.com/thrive/tracks/dynamic-programming)

### **Module 11: Greedy Algorithms**
*   **Topics:** Standard greedy problems (Activity Selection, Fractional Knapsack), understanding greedy-choice property.
*   **Resources:** [GeeksforGeeks: Greedy Algorithms](https://www.geeksforgeeks.org/greedy-algorithms/)

### **Module 12: Basic Number Theory & Math**
*   **Topics:** Prime numbers (Sieve of Eratosthenes), GCD/LCM, Modular Arithmetic.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/)

---

## Part 3: Advanced & Expert Topics

This section covers advanced topics required for top-tier performance.

### **Module 13: Advanced Data Structures**
*   **Topics:** Segment Tree, Fenwick Tree (BIT), Sparse Table, Tries.
*   **Advanced:** Persistent Segment Trees, Treaps / Splay Trees.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/), USACO Guides

### **Module 14: Advanced Graph Algorithms**
*   **Topics:** Shortest Path (Dijkstra, Bellman-Ford, Floyd-Warshall), Minimum Spanning Tree (Kruskal, Prim), Strongly Connected Components (SCC), Lowest Common Ancestor (LCA).
*   **Advanced:** Maximum Flow (Edmonds-Karp, Dinic), 2-SAT.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/)

### **Module 15: Advanced Dynamic Programming**
*   **Topics:** DP on Trees, DP with Bitmasking, Digit DP.
*   **Advanced:** Convex Hull Trick, DP optimizations (Knuth-Yao).
*   **Resources:** Codeforces Blogs, CSES DP section

### **Module 16: String Algorithms**
*   **Topics:** Knuth-Morris-Pratt (KMP), Z-Algorithm, Hashing.
*   **Advanced:** Suffix Arrays, Suffix Automaton, Aho-Corasick.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/)

### **Module 17: Computational Geometry**
*   **Topics:** Points, lines, vectors, Convex Hull, Sweep-line algorithms.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/)

### **Module 18: Advanced Mathematical Concepts**
*   **Topics:** Combinatorics (nCr, Catalan numbers), Probability, Game Theory (Sprague-Grundy Theorem).
*   **Advanced:** Fast Fourier Transform (FFT) for polynomial multiplication.
*   **Resources:** [CP-Algorithms](https://cp-algorithms.com/), Project Euler

### **Module 19: Theoretical Computer Science**
*   **Topics:** Amortized Analysis, Automata and Grammars (Finite State Machines).
*   **Resources:** CLRS book, Codeforces Blogs

### **Module 20: Miscellaneous Techniques**
*   **Topics:** Square Root Decomposition, Mo's Algorithm on Trees.
*   **Resources:** Codeforces Blogs

---

## Practice Strategy

*   **Beginner:** Focus on solving problems from the CSES Problem Set and Codeforces Div. 3 & 4.
*   **Intermediate:** Move to Codeforces Div. 2, AtCoder Beginner/Regular Contests, and USACO Bronze/Silver.
*   **Advanced/Expert:** Compete in Codeforces Div. 1, AtCoder Regular/Grand Contests, TopCoder SRMs, and USACO Gold/Platinum. Regularly participate in virtual contests and upsolve problems.
